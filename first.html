What's the Difference?

HTML

Hypertext Markup Language
Create the structure
  • control the layout of the content
  • Provides structure for web page design
  • The fundamental building block of any web page design

CSS

Cascading Style Sheet
Stylize the website
  • Applies style to the web page elements
  • Target various screen sizes to make web page responsive
  • primarily handles the "look and feel"of web page

Javascript

Increase interactivity
  • Adds interactivity to a web page
  • Handles complex functions and features
  • Programatic code which enhances functionality
